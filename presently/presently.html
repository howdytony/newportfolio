Context

Presently is a birthday reminder and gift-suggestion app that was created in response to a hackathon theme of automation, with the intention of building a product that would be immediately useful in people’s lives.

We asked ourselves: "How might we help caring individuals stay on top of their friend’s birthdays and get them meaningful, unique gifts?" To that end we conducted competitive analysis and learned that while there are quite a few birthday reminder apps on the marketplace, none gave truly rich, tailored suggestions. We built a clickable prototype and style guide in Figma that the developers could reference while building feature functionality and styling CSS for the deployed app.

Concept and Visual Direction

Gift-giving should be fun–not a chore. We wanted to lean into this with Presently’s visual design. We landed on a colorful, hand-built clay aesthetic that calls to mind the human touch of perfectly-picked gift, and a cheerful, celebratory color palette that reflects the joy of a party.

Usability Testing

We tested Presently's design for the successful completion of tasks including

  • Creating an account
  • Adding a new friend profile (add & edit tags)
  • Generating gift recommendation based on tags

Users were able to complete tasks with no major blockers; only minor pain points addressed below. Overall feedback was positive, with one user saying “This is a fantastic idea! I would download this right now.”

Visibility of System Status
Users needed clarification about the clickability of the page toggles on the friend profile screen, so we upped the contrast to follow recommended 7:1 ratio.

Hierarchy of Scale
Users expressed confusion about the layout of the friend button, feeling it overlapped too significantly with the profile cards.

Consistency, Standards, and Error Prevention
Users expressed needing clarification on which sections of the signup process were mandatory and the correct format for passwords/dates.
